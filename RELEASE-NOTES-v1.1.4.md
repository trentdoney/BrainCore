# BrainCore v1.1.4 Release Notes

This release completes the public launch surface for BrainCore.

Highlights:

- public README now centers the actual launch proof strip
- launch docs are normalized to `001` through `008` and a
  `14-table preserve schema`
- benchmarks separate smoke-regression artifacts from production-corpus
  artifacts
- the example MCP server remains a minimal reference integration
- the repository ships BrainCore-specific contribution and issue
  templates

Launch proof:

- `26,966 facts`
- `P50 71.6 ms`
- `P95 85.2 ms`
- `98.52% evidence grounding`

Smoke-regression proof remains committed separately and should not be
used as a headline claim.
